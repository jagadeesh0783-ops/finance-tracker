Cloud Sync (Firebase)

Auto-sync across all your devices. Just paste your Firebase database URL.

📋 One-Time Setup (1 minute)
  1. Go to console.firebase.google.com
  2. Click Create a project → Name it anything → Skip analytics → Create
  3. In sidebar click Build → Realtime Database
  4. Click Create Database → Pick any location → Start in test mode → Enable
  5. Copy the URL at the top (looks like https://xxx-default-rtdb.firebaseio.com)
  6. Paste it above — done! ✅
